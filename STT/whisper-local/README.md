# AI-Powered-Chatbots-and-Virtual-Agents-for-Mobitel-Contact-Center-
Mobitel aims to improve customer service while reducing dependency on human agents. This
project proposes the use of AI-powered chatbots and virtual agents that can handle customer
queries across Sinhala, English, and Tamil.
By automating repetitive and common tasks, Mobitel can improve efficiency, reduce costs, and
provide faster responses to customers.
